# Python Practice Repository

This repository contains beginner Python programs while learning Python.

Topics covered:
- loops
- conditions
- strings
- lists
- palindrome checker
- prime numbers
- factorial
- perfect numbers
- leap year checker
- multiplication tables

Goal:
To practice Python consistently and improve problem-solving skills. 

 Codes may not be one of those professional codes or something but the logic is right and also the codes work correctly.
